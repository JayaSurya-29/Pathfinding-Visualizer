A visual tool to demonstrate how different pathfinding algorithms work on a grid.
This project helps users understand how algorithms explore nodes and find the shortest path between two points.

🚀 Features
Visual representation of pathfinding algorithms
Interactive grid
Start and end node selection
Wall/obstacle creation
Real-time algorithm visualization

🧠 Algorithms Implemented
Breadth First Search (BFS)
Depth First Search (DFS)
Dijkstra's Algorithm

Each algorithm demonstrates a different strategy for exploring nodes and finding paths.

🛠️ Technologies Used
JavaScript – Algorithm visualization
HTML – User interface structure
CSS – Styling and animations
Java – Algorithm implementations
Gradle – Build configuration
